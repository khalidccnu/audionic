### Getting Started
<p>Firstly, clone this repository on your local machine using below command:</p>
<code>git clone https://github.com/khalidccnu/audionic.git</code>

<p>Secondly, in the project directory, you need to install dependencies. So, run below command:</p>
<code>npm install</code>

<p>Now, you need to create environment variables in project directory, which are named below:</p>
<code>VITE_RAPID_API_KEY="Your Rapid Spotify API Key"</code>
<br>
<code>VITE_KC_URL="Your Keycloak URL"</code>
<br>
<code>VITE_KC_RM="Your Keycloak Realm"</code>
<br>
<code>VITE_KC_CID="Your Keycloak Client ID"</code>

Finally, if you want to see screenshot, then follow [link](https://github.com/khalidccnu/audionic/tree/main/screenshot).
